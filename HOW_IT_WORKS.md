\# 智慧命題 2.0 - 系統使用說明



\## 系統運作方式



\### 需要 2 個步驟：



1\. \*\*你先準備 50-100 題示範題目\*\*（只做一次）

&nbsp;  - 這是訓練資料，讓系統學習什麼是好題目



2\. \*\*系統學會後，就能自動生成無限題目\*\*

&nbsp;  - 只需告訴它考科和難度



\## 完整流程

```

準備 50-100 題示範 → 訓練模型（5分鐘）→ 無限生成題目

```



\## 快速開始



\### 第一次使用（1-2 小時）

```python

\# 1. 準備訓練資料

training\_data = \[

&nbsp;   {"question": "題目...", "quality\_score": 9.0},

&nbsp;   # ... 50-100 題

]



\# 2. 訓練模型

python src/train\_lasso\_model.py



\# 3. 完成！

```



\### 之後使用（30 秒）

```python

questions = system.generate\_and\_filter(

&nbsp;   topic="憲法", 

&nbsp;   count=20

)

```
